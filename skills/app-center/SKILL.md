# Skill: 应用中心

## 触发条件
用户说类似以下的话时触发此技能：
- "打开应用中心"
- "有哪些安装的应用"
- "查看已安装的应用"
- "应用列表"
- "怎么访问安装的应用"

## 执行逻辑

### 1. 扫描已安装应用

```bash
ls -d /root/.openclaw/apps/*/app.json 2>/dev/null | while read f; do
  cat "$f"
  echo "---"
done
```

### 2. 回复用户

根据扫描结果，向用户展示已安装的应用列表：

**回复模板：**

```
📦 已安装的应用：

{对于每个应用}
{icon} **{displayName}** (v{version})
  {description}
  🔗 入口：{entryPath}
  状态：{running/stopped}
  功能：{features}

---

💡 你可以在管理面板的「应用中心」标签页中直接点击访问这些应用。
💡 想安装新应用？告诉我应用名称，我会帮你自动安装。
```

### 3. 如果没有已安装的应用

```
📦 当前没有安装任何应用。

可安装的应用：
⚛️ **江苏高中物理知识网站** — 涵盖初中到高中全部物理知识
   安装命令：「帮我安装物理知识网站」

想要安装某个应用？直接告诉我即可。
```

## 重要提示

- 所有已安装的应用都在管理面板的「应用中心」页面中可以看到
- 用户应该从应用中心页面点击链接来访问应用，而不是记住 URL
- 安装新应用时，需要按照各应用的 SKILL.md 中的指令执行
- 应用的入口路径（entryPath）是相对于管理面板域名的
